# Atelier UI Kit

A public "lab view" of RBX's systems — the atelier functions as a living index of experiments, telemetry and activity.

**Screens / sections**
- `AtelierHero` — status pill + brass accent line (accent reserved for "institutionalized" / live state)
- `AtelierMetrics` — 4-up telemetry card strip (Active, Institutionalized, Commits, Uptime)
- `AtelierIndex` — tabular product index with mono metrics + hover reveal
- `AtelierTimeline` — recent activity feed with status pills

**Components used**
All `_shared` primitives + `TopNav`, `Footer`.
